Hello There !
-------------

*The project is implemented using Selenium with Java language also Maven and Cucumber are used and the report used is the extent report.

This is an assignment for testing symfonycasts.com APIs with a given scenario 

1. Verify that farmer can get valid token with valid ClientSecret or blocked with invalid client secret.
2. Verify that farmer Can/Can't Unlock the barn using valid/invalid key.
3. Verify that farmer Can/Can't use the restroom using valid/invalid key.
4. Verify that farmer Can/Can't feed the chicken using valid/invalid key.
... to be continued

How to run ! 
--------------
To run the assignment scenario there are three possible ways 

1- From Testng.xml Go to "lieferando/src/test/java/TestRunner/testng.xml" and run it.

2- From TestRunner.xml Go to "lieferando/src/test/java/TestRunner/TestRunner.xml" and run it.

3- Go to the implemented scenario feature file "lieferando/src/test/java/FeatureFile/API/Task.feature" and run it

4- For any added scenarios specify tag for that TC and pass it to the test runner in "tag" value .
